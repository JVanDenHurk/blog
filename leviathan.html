Leviathan CTF Walkthrough

Part of the OverTheWire wargames — beginner-friendly and full of cool little tricks. Here's how I solved each level.

leviathan0

ls -la to find hidden files.
Found bookmarks.html.
Opened it with less and searched for "password". Found the password for leviathan1.

Flag: 3QJ3TgzHDq

leviathan1

Found a file called check, but had no access at first.
Things I tried:

  • cat → got a wall of nonsense
  • strings → found “secr” and “love”
  • ./check → asked for password; tried “secr”, “love”, “secrlove” → failed
  • ltrace ./check → revealed strcmp() call; showed password was sex

Switched to user leviathan2 and grabbed password from /etc/leviathan_pass.

Flag: NsN1HwFoyN

leviathan2

Provided binary tried to print files but blocked paths like ../leviathan3/level3 and /etc/leviathan_pass/leviathan3.

Solution:

  • Created a symlink in /tmp → ln -s /etc/leviathan_pass/leviathan3 hacked
  • touch "read.txt" to spoof the second argument
  • ./printfile "hacked read.txt" tricked the program into reading the symlinked file

Flag: f0n8h2iWLP

leviathan3

Ran ltrace ./level3, revealed:

strcmp("h0no33", "snlprint")

Entered snlprint and it worked.

Flag: WG1egElCvO

leviathan4

ls -la revealed hidden .trash directory. Inside it was a file called bin.

Ran it — it dumped binary. Used an online binary-to-text translator to decode.

Flag: 0dyxT7F4QD

leviathan5

Binary read from /tmp/file.log.

Created a symlink:

ln -s /etc/leviathan_pass/leviathan6 /tmp/file.log
./leviathan5

Flag: szo7HDB88w

leviathan6

Binary asked for a 4-digit PIN. Wrote a brute-force script:

#!/bin/bash
for a in {0000..9999}
do
  ./leviathan6 $a
done
