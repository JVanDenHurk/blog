TryHackMe Pre-Security Certification

The TryHackMe Pre-Security pathway gave me foundational knowledge in cybersecurity, covering the World Wide Web, Linux, Windows and networking fundamentals.

Intro to Cybersecurity

This module introduced core concepts of cybersecurity, including the difference between offensive and defensive security.

Network Fundamentals

Developed an understanding of how computers communicate within networks. Key topics included:

  • Networking Concepts: Understanding IP addresses, ports, protocols and how data moves through networks.
  • Network Devices: Functions of switches, routersand firewalls.
  • Security: Analyzing packets, frames and firewall configurations.

World Wide Web Fundamentals

Gained insights into the architecture and functioning of the World Wide Web. Covered:

  • Web Protocols: HTTP, HTTPS and status codes.
  • Web Technologies: HTML, JavaScript, cookies and headers.
  • DNS and Domain Hierarchy: Understanding how domain names are resolved.

Linux Fundamentals

Learned to manage and operate Linux systems. Focused on:

  • Shell and Commands: Navigating the file system and executing scripts.
  • File System: Managing files and permissions effectively.
  • Package Management: Installing and maintaining software using apt and yum.
  • System Logs: Monitoring and analyzing log files.

Windows Fundamentals

Developed a foundational understanding of Windows system management. Covered:

  • System Management: Using the control panel, task manager and resource monitor.
  • Command Line Tools: Managing systems with the command prompt and PowerShell.
  • Security Features: BitLocker, virus protection and user account management.
