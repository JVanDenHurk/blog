Windows and AD Fundamentals

Gained foundational knowledge of the Windows operating system and how Active Directory (AD) is used to manage users, computers, and permissions in a networked environment. Key topics included:

  • Windows OS Basics
    Understanding the Windows file system, command line (CMD and PowerShell), user roles, and system tools.
  • Active Directory
    Exploring domain controllers, organizational units (OUs), users, groups, and group policy management.
  • Windows Security
    Learning about access control, user authentication, and common security configurations in a Windows environment.

Command Line

Gained comprehensive hands-on experience with the Linux shell and Windows PowerShell, learning how to interact with operating systems, automate tasks, and analyze system behavior. Key topics included:

  • Linux Shell Basics
    Introduction to Linux command-line interfaces (CLIs), including bash and other shell types. Practiced using commands to navigate, manipulate files, and execute scripts.
  • System & Network Information
    Retrieved system info using commands like uname, top, ps, ifconfig/ip, and used tools like ping, netstat, and traceroute for basic network troubleshooting.
  • File & Disk Management
    Learned to manage files, directories, and disk space using tools such as df, du, lsblk, and mount.
  • Process & Task Management
    Monitored and controlled system processes using ps, top, kill, and job control commands like bg and fg.
  • Shell Scripting
    Gained exposure to basic shell scripting concepts—variables, loops, conditionals, and writing practical scripts like the "Locker Script" for automation.
  • PowerShell Fundamentals
    Explored Windows PowerShell, learning how to work with cmdlets, retrieve system and network data, and perform file system operations.
  • Piping, Filtering, and Sorting
    Mastered data manipulation by chaining commands using pipes and cmdlets like Sort-Object, Where-Object, and Select-Object in PowerShell.
  • Real-Time Analysis & Automation
    Used PowerShell for live system monitoring, real-time process analysis, and wrote scripts to automate repetitive administrative tasks.

Networking Fundamentals

Networking is the backbone of cybersecurity, and this section gave me a strong foundation in how devices communicate, how data travels, and how we can analyze and secure that communication. From basic models to real-world tools, I explored how everything connects.

Networking Concepts

  • OSI & TCP/IP Models
    Got to grips with how data moves through different layers of a network, from physical wires all the way up to applications.
  • IP Addresses & Subnets
    Learned how devices are assigned unique addresses and how subnetting helps organize and secure networks.
  • TCP vs UDP
    Explored the differences between reliable (TCP) and fast (UDP) protocols and when to use each.
  • Encapsulation
    Understood how data is packaged and passed through layers to reach its destination.
  • Telnet
    Used Telnet to connect to remote machines and saw how insecure protocols work under the hood.

Networking Essentials

  • DHCP
    Discovered how devices automatically receive their IP settings when they join a network.
  • ARP
    Learned how IP addresses are linked to physical MAC addresses so devices can talk locally.
  • ICMP
    Used tools like ping and tracert to diagnose network issues and understand how packets are routed.
  • Routing & NAT
    Explored how routers direct traffic between networks, and how NAT translates private IPs to public ones.

Core Networking Protocols

  • DNS
    Understood how human-friendly domain names get translated into IP addresses computers can use.
  • WHOIS
    Used WHOIS lookups to gather information about domain ownership and registration details.
  • HTTP/HTTPS
    Learned how websites are served over the internet and how HTTPS protects web traffic.
  • FTP, SMTP, POP3, IMAP
    Explored older protocols for file transfer and email, including how and why they can be insecure if not encrypted.

Secure Networking Protocols

  • TLS & HTTPS
    Dived into how encryption secures web traffic, and how TLS keeps data safe in transit.
  • SSH
    Used SSH to securely connect to remote systems via the command line — a must-have skill for sysadmins and pentesters.
  • SFTP, FTPS, SMTPS, POP3S, IMAPS
    Looked at secure versions of file and email transfer protocols, and how encryption is layered on top of legacy systems.
  • VPNs
    Explored how VPNs create encrypted tunnels for safe browsing and private communication, even over public Wi-Fi.

Wireshark: The Basics

  • Capturing Packets
    Used Wireshark to capture live traffic and understand how data looks as it flows through a network.
  • Packet Dissection
    Dug into individual packets to identify protocols, payloads, and potential issues.
  • Filtering Traffic
    Applied filters to zero in on specific conversations or protocols — super helpful when troubleshooting.

Tcpdump: The Basics

  • Command-Line Capture
    Learned to capture network traffic using just the terminal, a lighter alternative to Wireshark.
  • Filtering Expressions
    Used precise filter syntax to capture only the traffic I wanted to analyze.
  • Display Options
    Customized output formats to make it easier to interpret results directly from the CLI.

Nmap: The Basics

  • Host Discovery
    Scanned for live systems in a network to map out the environment.
  • Port Scanning
    Identified open ports to figure out what services are running — crucial for vulnerability assessments.
  • Version Detection
    Pulled back banner information and software versions to evaluate risk and exposure.
  • Scan Timing & Output
    Tuned scan speeds and output formats for stealth or performance depending on the situation.
