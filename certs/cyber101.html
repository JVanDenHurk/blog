Windows and AD Fundamentals

Gained foundational knowledge of the Windows operating system and how Active Directory (AD) is used to manage users, computers, and permissions in a networked environment. Key topics included:

  • Windows OS Basics
    Understanding the Windows file system, command line (CMD and PowerShell), user roles, and system tools.
  • Active Directory
    Exploring domain controllers, organizational units (OUs), users, groups, and group policy management.
  • Windows Security
    Learning about access control, user authentication, and common security configurations in a Windows environment.

Command Line

Gained comprehensive hands-on experience with the Linux shell and Windows PowerShell, learning how to interact with operating systems, automate tasks, and analyze system behavior. Key topics included:

  • Linux Shell Basics
    Introduction to Linux command-line interfaces (CLIs), including bash and other shell types. Practiced using commands to navigate, manipulate files, and execute scripts.
  • System & Network Information
    Retrieved system info using commands like uname, top, ps, ifconfig/ip, and used tools like ping, netstat, and traceroute for basic network troubleshooting.
  • File & Disk Management
    Learned to manage files, directories, and disk space using tools such as df, du, lsblk, and mount.
  • Process & Task Management
    Monitored and controlled system processes using ps, top, kill, and job control commands like bg and fg.
  • Shell Scripting
    Gained exposure to basic shell scripting concepts‚Äîvariables, loops, conditionals, and writing practical scripts like the "Locker Script" for automation.
  • PowerShell Fundamentals
    Explored Windows PowerShell, learning how to work with cmdlets, retrieve system and network data, and perform file system operations.
  • Piping, Filtering, and Sorting
    Mastered data manipulation by chaining commands using pipes and cmdlets like Sort-Object, Where-Object, and Select-Object in PowerShell.
  • Real-Time Analysis & Automation
    Used PowerShell for live system monitoring, real-time process analysis, and wrote scripts to automate repetitive administrative tasks.
